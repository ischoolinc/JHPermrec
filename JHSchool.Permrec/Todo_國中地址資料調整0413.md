## 一、問題摘要

目前 `AddressPalmerwormItem.cs` 在 `BindDataToForm()` 載入學生地址資料時：

1. 先把資料庫內的 `ZipCode` 指定給 `txtZipcode.Text`
2. 再指定 `cboCounty.Text`
3. 再指定 `cboTown.Text`

但 `cboTown.Text` 在初始化綁值時，會觸發：

- `cboTown_TextChanged()`
- `CheckTownChange()`

而 `CheckTownChange()` 會依目前縣市/鄉鎮市區對照表，反向改寫 `txtZipcode.Text`。

因此會發生：
- 資料原本 `txtZipcode = 312`
- 畫面載入後，又被 `County + Town` 對照改成 `308`

這是初始化事件誤觸發造成的，不是使用者手動修改造成的。

---

## 二、修改目標

只修正這件事：

- 在 `BindDataToForm()` 綁定畫面資料期間，暫停地址欄位事件造成的反向同步
- 讓 `txtZipcode` 保留資料庫原始值
- 等畫面綁定完成後，再恢復正常事件行為

注意：
- 不可改壞原本使用者手動修改縣市、鄉鎮市區時，自動回填郵遞區號的功能
- 不可改壞原本輸入郵遞區號後，自動反查縣市、鄉鎮市區的功能
- 不可修改 Designer 檔
- 不可改動 Save 流程與資料表欄位寫回邏輯

## 五、實作步驟

### 1. 調整 BindDataToForm() 的初始化保護範圍

在 `BindDataToForm()` 內，真正開始把 `_StudAddressRec` 的值指定到畫面控制項前：

- 先設定 `_date_updating = true;`

等該次地址資料（戶籍 / 聯絡 / 其它）全部欄位都指定完成後：

- 再設定 `_date_updating = false;`

### 2. 保護範圍要涵蓋以下欄位綁值

不論是：

- Permanent
- Mailing
- Other

都要確保以下欄位綁值期間在保護範圍內：

- `txtZipcode.Text`
- `cboCounty.Text`
- `cboTown.Text`
- `txtDistrict.Text`
- `txtArea.Text`
- `txtDetail.Text`
- `txtLongtitude.Text`
- `txtLatitude.Text`

### 3. 要用 try/finally 包住

避免綁值中途發生例外時， `_date_updating` 永遠卡在 `true`。

請使用這種安全模式：

```csharp
_date_updating = true;
try
{
    // 畫面欄位綁值
}
finally
{
    _date_updating = false;
}
````

### 4. 不要改動下列方法原本邏輯

以下方法邏輯保持原樣：

* `CheckTownChange()`
* `CheckZipCode()`
* `cboTown_TextChanged()`
* `txtZipcode_KeyDown()`
* `txtZipcode_Validated()`
* `OnSaveButtonClick()`

原因：
本次只修初始化載入時事件誤觸發，不修正常互動邏輯。

## 六、建議修改方式

### A. BindDataToForm() 修改原則

請將 `BindDataToForm()` 內各地址類型資料指定區塊，包進 `_date_updating = true/false` 保護中。

#### 戶籍地址

現在類似：

```csharp
txtZipcode.Text = _StudAddressRec.Permanent.ZipCode;
cboCounty.Text = _StudAddressRec.Permanent.County;
cboTown.Text = _StudAddressRec.Permanent.Town;
...
```

請改成安全模式：

```csharp
_date_updating = true;
try
{
    txtZipcode.Text = _StudAddressRec.Permanent.ZipCode;
    cboCounty.Text = _StudAddressRec.Permanent.County;
    cboTown.Text = _StudAddressRec.Permanent.Town;
    txtDistrict.Text = _StudAddressRec.Permanent.District;
    txtArea.Text = _StudAddressRec.Permanent.Area;
    txtDetail.Text = _StudAddressRec.Permanent.Detail;
    txtLongtitude.Text = _StudAddressRec.Permanent.Longitude;
    txtLatitude.Text = _StudAddressRec.Permanent.Latitude;
}
finally
{
    _date_updating = false;
}
```

#### 聯絡地址、其它地址

同樣比照辦理。
